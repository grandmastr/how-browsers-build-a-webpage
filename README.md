# How Browsers Build A Webpage

This is a guide I decided to put together while diving deeper into web performance, optimizations/improvements provided
by modern rendering engines, and some of the right choices to make when optimizing a web application.

It includes notes on how browsers build a webpage. It's aimed at developers who need quick references and refreshers on
the major parts of the process.

The outline below walks through the steps involved in building a webpage and how each relates to web performance.
